### Running the Program

Ensure you're in the project root (Rafid-Nasery-OS-P3)

#### First things first run:
```bash
javac ./src/project3.java
```
#### General format for commands
```bash
java ./src/project3.java <command> [arguments...]
```

#### Example commands
```bash
java ./src/project3.java print test.idx
java ./src/project3.java search test.idx 15
```
For examples on output please look at my testing in the testing-terminal.txt file
